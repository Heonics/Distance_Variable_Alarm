Required Materials :
1 X Arduino Uno
1 X HC-SR04
1 X 5V Buzzer
1 X Breadboard(optional)

Circuit Diagram :
Arduino Digital Pin 13 -> Buzzer +ve
Arduino GND -> Buzzer -ve
Arduino 5V -> HC-SR04 +ve
Arduino GND -> HC-SR04 +ve
Arduino Digital Pin 12 -> HC-SR04 Echo
Arduino Digital Pin 13 -> HC-SR04 Trig
